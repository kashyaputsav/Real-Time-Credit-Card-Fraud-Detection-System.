# 💳 Real-Time Credit Card Fraud Detection System

## 🚀 Overview
End-to-end fraud detection system using XGBoost, SMOTE, and SHAP explainability with real-time API deployment.

---

- Precision: **93%**
- Recall: **85%**
- Optimized using precision-recall tradeoff

- Threshold optimization
- SHAP explainability
- FastAPI deployment
- Frontend UI
